# Day 1 - Band Name Generator

## Description

This is a simple Python program that generates a band name using:

* The city you grew up in
* Your pet's name

## What I Learned

* print() for output
* input() for user input
* variables
* string formatting with f-strings

## How to Run

Run the program and follow the prompts.

## Example

Input:
Tokyo
Orion

Output:
Tokyo Orion
