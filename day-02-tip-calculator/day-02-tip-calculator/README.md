# Day 2 - Tip Calculator

## Description

This program calculates how much each person should pay when splitting a bill, including a tip.

## What I Learned

* Data types (int, float)
* Mathematical operations
* Converting user input
* Using round() for formatting

## How to Run

1. Enter the total bill amount
2. Choose a tip percentage
3. Enter number of people
4. Get the amount each person should pay

## Example

Input:
Total bill: 150
Tip: 12
People: 5

Output:
Each person should pay: $33.6
